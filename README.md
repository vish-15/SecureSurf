# Website Content Analyzer

This project is a web application designed to analyze the content of websites. It allows users to input a URL, fetches the website's content, and then leverages the power of the Gemini API to analyze the text, presenting the results in a clear and user-friendly manner. Built with TypeScript and utilizing the Shadcn/ui component library, this application provides a robust and intuitive platform for website content analysis.

**Key Features:**

*   **URL Input:** Users can input any valid URL to analyze its content.
*   **Content Fetching:** The application fetches the content of the specified website.
*   **AI-Powered Analysis:** Employs the Gemini API to analyze the fetched content, providing insights and summaries.
*   **User-Friendly Presentation:** Presents the analysis results in a structured and easily understandable format, leveraging the Shadcn/ui component library for a clean interface.
*   **Full-Stack:** This is a full stack application.

**Technical Highlights:**

*   **Full-Stack Development:** This project encompasses both frontend and backend development for a complete web application.
*   **TypeScript:** The entire application is built using TypeScript, ensuring type safety and enhanced code quality.
*   **Gemini API Integration:** Integrates with the Gemini API to provide AI-powered content analysis.
*   **Shadcn/ui:**  Utilizes the Shadcn/ui component library for a modern and customizable user interface.
* **AI-Powered Content Analysis:** The core strength of this project is the ability to understand and summarize website content using AI.

**Project Structure:**

*   `src/ai`: Contains the core AI logic and algorithms used for content analysis.
*   `src/components/ui`: Houses all the Shadcn/ui components, which ensure a modern user interface.
*   `src/app/api`: Contains the API route responsible for fetching the website content.
*   `src/services`: Contains functions for interacting with the Gemini API, including sending requests and processing responses.

**Getting Started:**

This project is built using Next.js and Firebase Studio. To start exploring the project, take a look at `src/app/page.tsx`.

**Note**:  Before using this app make sure you configure all the requiered enviroment variables.
